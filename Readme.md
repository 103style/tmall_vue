# Tmall Web

## 运行项目
```
> git clone https://github.com/103style/tmall_vue.git
> cd tmall_vue
> npm i
> npm run dev
```

---

## build相关
通过 `npm run build` 命令打包。

不过由于图片资源去掉了协议头， 所以本地打开会看不到一些图片。
